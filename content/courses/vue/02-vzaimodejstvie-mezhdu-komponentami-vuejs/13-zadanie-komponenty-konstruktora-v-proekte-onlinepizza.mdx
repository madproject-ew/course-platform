# Задание: компоненты конструктора в проекте OnlinePizza

В этом задании вы вынесете логику конструктора пиццы из `HomeView.vue` в отдельные компоненты. Каждый шаг конструктора станет самостоятельным компонентом со своими пропсами и событиями. Примените знания о взаимодействии между компонентами из предыдущих уроков.

## Шаг 1. Создайте директорию модуля

Создайте директорию `src/modules/constructor/`. Все компоненты конструктора будут размещены здесь.

## Шаг 2. Создайте DoughSelector.vue

Создайте файл `src/modules/constructor/DoughSelector.vue`.

Этот компонент отвечает за выбор типа теста.

- **Пропсы:**
  - `items` (тип Array) — массив доступных типов теста (объекты с полями `id`, `name`, `description`, `price`)
  - `modelValue` (тип Object) — текущий выбранный тип теста

- **Событие:**
  - `update:modelValue` — генерируется при выборе нового типа теста

- **Шаблон:**
  - Заголовок секции "Тесто"
  - Список элементов выбора, отрисованных через `v-for`. Каждый элемент отображает название и описание типа теста
  - Выбранный элемент выделяется с помощью динамического класса (сравнение `item.id` с `modelValue.id`)
  - При клике на элемент генерируется событие `update:modelValue` с выбранным объектом

По сути, вы переносите в этот компонент секцию выбора теста из `HomeView.vue`, но теперь данные приходят через пропсы, а выбор сообщается через событие.

## Шаг 3. Создайте SizeSelector.vue

Создайте файл `src/modules/constructor/SizeSelector.vue`.

Этот компонент отвечает за выбор размера пиццы.

- **Пропсы:**
  - `items` (тип Array) — массив размеров (объекты с `id`, `name`, `multiplier`)
  - `modelValue` (тип Object) — текущий выбранный размер

- **Событие:**
  - `update:modelValue` — при выборе нового размера

- **Шаблон:**
  - Заголовок "Размер"
  - Элементы выбора в виде группы кнопок, расположенных в ряд. Активная кнопка выделена стилем
  - При клике генерируется событие с выбранным объектом

## Шаг 4. Создайте SauceSelector.vue

Создайте файл `src/modules/constructor/SauceSelector.vue`.

Компонент для выбора соуса.

- **Пропсы:**
  - `items` (тип Array) — массив соусов (объекты с `id`, `name`, `price`)
  - `modelValue` (тип Object) — текущий выбранный соус

- **Событие:**
  - `update:modelValue` — при выборе нового соуса

- **Шаблон:**
  - Заголовок "Соус"
  - Элементы выбора, аналогичные `DoughSelector` — выбранный элемент выделяется динамическим классом

## Шаг 5. Создайте IngredientsSelector.vue

Создайте файл `src/modules/constructor/IngredientsSelector.vue`.

Компонент для выбора ингредиентов и их количества.

- **Пропсы:**
  - `items` (тип Array) — массив ингредиентов (объекты с `id`, `name`, `price`, `category`)
  - `modelValue` (тип Object) — объект с количествами в формате `{ [ingredientId]: quantity }`

- **Событие:**
  - `update:modelValue` — генерируется при изменении количества любого ингредиента

- **Шаблон:**
  - Заголовок "Ингредиенты"
  - Сетка карточек ингредиентов через `v-for`
  - В каждой карточке: название, цена и компонент `AppCounter` (созданный в предыдущем задании)
  - `AppCounter` связывается с соответствующим значением из `modelValue` по ключу `item.id`

- **Логика:**
  - При изменении значения `AppCounter` создайте копию объекта `modelValue`, обновите нужное значение и сгенерируйте событие `update:modelValue` с новым объектом

## Шаг 6. Создайте PizzaPreview.vue

Создайте файл `src/modules/constructor/PizzaPreview.vue`.

Компонент для отображения текущей конфигурации пиццы.

- **Пропсы:**
  - `dough` (тип Object) — выбранный тип теста
  - `size` (тип Object) — выбранный размер
  - `sauce` (тип Object) — выбранный соус
  - `ingredients` (тип Array) — массив выбранных ингредиентов (объекты с полями из ингредиента + `quantity`)

- **Шаблон:**
  - Карточка-превью, отображающая текстовую сводку текущей пиццы:
    - Тесто: название выбранного теста
    - Размер: название выбранного размера
    - Соус: название выбранного соуса
    - Ингредиенты: список добавленных ингредиентов с количеством (только те, у которых количество больше 0)
    - Если ингредиенты не выбраны — текст "Ингредиенты не выбраны"

- **Вычисляемое свойство (computed):**
  - Рассчитайте базовую стоимость пиццы: (цена теста + цена соуса + сумма цен ингредиентов с учётом количества) * множитель размера. Отобразите итоговую цену в превью

Все компоненты должны находиться в директории `src/modules/constructor/`.

## Проверка

На этом этапе компоненты пока не подключены к странице — это будет в следующем задании. Но убедитесь, что:

- Все файлы созданы в правильной директории
- В каждом компоненте определены пропсы и события
- Шаблоны компонентов содержат корректную разметку
- Нет синтаксических ошибок (проверьте, что проект запускается без ошибок)
