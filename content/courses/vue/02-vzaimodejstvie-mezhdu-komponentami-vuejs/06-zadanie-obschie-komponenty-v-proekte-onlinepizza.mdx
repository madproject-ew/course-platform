# Задание: общие компоненты в проекте OnlinePizza

В этом задании вы создадите переиспользуемые компоненты, которые понадобятся в разных частях приложения. Примените знания о пропсах, событиях и `v-model` из предыдущих уроков.

## Шаг 1. Создайте компонент AppCounter

Создайте файл `src/common/components/AppCounter.vue`.

Этот компонент управляет числовым значением — увеличивает или уменьшает его. Он будет использоваться для выбора количества ингредиентов, а позже — для количества товаров в корзине.

Реализуйте следующее:

- **Пропсы:**
  - `modelValue` (тип Number) — текущее значение счётчика
  - `min` (тип Number, по умолчанию 0) — минимальное допустимое значение
  - `max` (тип Number, по умолчанию 10) — максимальное допустимое значение

- **Событие:**
  - `update:modelValue` — генерируется при изменении значения (это позволит использовать компонент с `v-model`)

- **Шаблон:**
  - Кнопка "-" для уменьшения значения. Должна быть заблокирована (`disabled`), когда значение равно `min`
  - Отображение текущего значения между кнопками
  - Кнопка "+" для увеличения значения. Должна быть заблокирована, когда значение равно `max`

- **Логика:**
  - При нажатии "-" уменьшите значение на 1 и сгенерируйте событие `update:modelValue` с новым значением
  - При нажатии "+" увеличьте значение на 1 и сгенерируйте событие
  - Не допускайте выхода за пределы `min` и `max`

## Шаг 2. Создайте компонент AppDrag

Создайте файл `src/common/components/AppDrag.vue`.

Этот компонент-обёртка делает любой вложенный элемент перетаскиваемым с помощью HTML5 Drag API.

Реализуйте следующее:

- **Пропсы:**
  - `transferData` (тип Object) — данные, которые будут переданы при перетаскивании

- **Шаблон:**
  - Корневой элемент (например, `<div>`) с атрибутом `draggable="true"`
  - Внутри — `<slot>` для вложенного содержимого

- **Логика:**
  - На событие `dragstart` корневого элемента повесьте обработчик, который записывает `transferData` в `event.dataTransfer` с помощью `setData('application/json', JSON.stringify(transferData))`

## Шаг 3. Создайте компонент AppDrop

Создайте файл `src/common/components/AppDrop.vue`.

Этот компонент-обёртка принимает перетаскиваемые элементы.

Реализуйте следующее:

- **Событие:**
  - `drop` — генерируется с переданными данными, когда элемент сброшен в эту область

- **Шаблон:**
  - Корневой элемент (например, `<div>`) с обработчиками событий `dragover` и `drop`
  - Внутри — `<slot>` для вложенного содержимого

- **Логика:**
  - На событие `dragover` вызовите `event.preventDefault()` — это разрешает сброс элемента в эту область
  - На событие `drop` вызовите `event.preventDefault()`, извлеките данные из `event.dataTransfer.getData('application/json')`, распарсите JSON и сгенерируйте событие `drop` с полученными данными

## Шаг 4. Подключите AppCounter к ингредиентам

Вернитесь в `src/views/HomeView.vue`.

В секции ингредиентов добавьте к каждой карточке компонент `AppCounter`. Подключите его через `v-model`, связав с соответствующим значением в объекте `ingredientQuantities` (по ключу `ingredient.id`).

Теперь у каждого ингредиента должны появиться кнопки "-" и "+" с отображением текущего количества.

## Проверка

Запустите проект и убедитесь, что:

- Компонент `AppCounter` отображается рядом с каждым ингредиентом
- Кнопка "+" увеличивает значение, "-" уменьшает
- Значение не опускается ниже 0 (кнопка "-" заблокирована при значении 0)
- Компоненты `AppDrag` и `AppDrop` созданы и не выдают ошибок (их полноценное использование мы проверим в следующих заданиях)
