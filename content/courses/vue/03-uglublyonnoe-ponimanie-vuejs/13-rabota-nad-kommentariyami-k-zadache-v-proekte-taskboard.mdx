# Шаг 1. Создание компонента комментариев

Создайте компонент `TaskCardViewComments.vue` в директории `src/modules/tasks/components`.

Начальная разметка компонента:

```html
<template>
  <div class="task-card__comments">
    <h2 class="task-card__title">
      Комментарии
    </h2>
    <div class="comments">
<!--      Список комментариев-->

<!--      Блок добавления нового комментария-->
    </div>
  </div>
</template>
```

# Шаг 2. Работа над списком комментариев

Добавим отображение комментариев:

```html
<!--      Список комментариев-->
<ul class="comments__list">
    <li
            v-for="comment in comments"
            :key="comment.id"
            class="comments__item"
    >
        <div class="comments__user">
            <img
                    :src="getImage(comment.user.avatar)"
                    :alt="comment.user.name"
                    width="30"
                    height="30"
            />
            {{ comment.user.name }}
        </div>
        <p>{{ comment.text }}</p>
    </li>
</ul>
```

Определим входные параметры компонента --- список комментариев и идентификатор задачи:

```html
<script setup>
import { getImage } from '@/common/helpers'

const props = defineProps({
  taskId: {
    type: Number,
    required: true
  },
  comments: {
    type: Array,
    default: () => []
  }
})
</script>
```

# Шаг 3. Добавление блока нового комментария

Реализуем поле ввода нового комментария с кнопкой отправки:

```html
      <!--      Блок добавления нового комментария-->
<form
        v-if="user"
        action="#"
        class="comments__form"
        method="post"
>
    <app-textarea
            v-model="newComment"
            name="comment_text"
            placeholder="Введите текст комментария"
            :error-text="validations.newComment.error"
    />
    <app-button
            class="comments__form__button"
            :type="'submit'"
            @click.prevent="submit"
    >
        Написать комментарий
    </app-button>
</form>
```

Импортируем компоненты, объявим реактивное свойство `newComment` и настроим его валидацию:

```html
<script setup>
import { ref } from 'vue'
import AppTextarea from '@/common/components/AppTextarea.vue'
import AppButton from '@/common/components/AppButton.vue'

const props = defineProps({
  taskId: {
    type: Number,
    required: true
  },
  comments: {
    type: Array,
    default: () => []
  }
})

const newComment = ref('')
const validations = ref({
  newComment: {
    error: '',
    rules: ['required']
  }
})
</script>
```

Для формирования объекта комментария нужен автор. На данном этапе мы возьмём первого пользователя из списка, а позднее заменим его на аутентифицированного пользователя:

```javascript
import { ref, computed } from 'vue'
import users from '@/mocks/users.json'

const user = computed(() => users[0])
```

Функция отправки комментария:

```javascript
import { validateFields } from '../../../common/validator'

const emits = defineEmits(['createNewComment'])

const submit = function () {
  // Проверяем, валидно ли поле комментария
  if (!validateFields({ newComment }, validations.value)) return
  // Создаём объект комментария
  const comment = {
    text: newComment.value,
    taskId: props.taskId,
    userId: user.value.id,
    user: {
      id: user.value.id,
      name: user.value.name,
      avatar: user.value.avatar
    }
  }
  // Отправляем комментарий в родительский компонент
  emits('createNewComment', comment)
  // Очищаем поле комментария
  newComment.value = ''
}
```

Когда пользователь начинает вводить новый текст после ошибки валидации, ошибку следует сбрасывать:

```javascript
import { ref, computed, watch } from 'vue'
import { validateFields, clearValidationErrors } from '../../../common/validator'

// Отслеживаем значение поля комментария и очищаем ошибку при изменении
watch(newComment, () => {
  if (validations.value.newComment.error) {
    clearValidationErrors(validations.value)
  }
})
```

# Шаг 5. Подключение компонента к TaskView

Интегрируем компонент комментариев в `TaskView` и добавим функцию добавления нового комментария в массив:

```html
<!--...-->
<!--Комментарии-->
  <task-card-view-comments
      v-if="task"
      class="task-card__comments"
      :comments="task.comments || []"
      :task-id="task.id"
      @create-new-comment="addCommentToList"
  />
<!--...-->
<script setup>
// ...
import TaskCardViewComments from '../modules/tasks/components/TaskCardViewComments.vue'

const addCommentToList = function (comment) {
  if (!task.value.comments) {
    task.value.comments = []
  }
  task.value.comments.push(comment)
}
//...
</script>
```
