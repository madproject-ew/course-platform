# Шаг 2. Закрытие диалога создания и редактирования задачи

Реализуем закрытие диалогового окна. Добавим обработчики событий в шаблон:

```html
<template>
  <div
      class="task-card"
      tabindex="0"
      ref="dialog"
      @click.self="closeDialog"
      @keydown.esc="closeDialog"
  >
    <section class="task-card__wrapper">
<!--      Кнопка закрытия диалога задачи-->
      <button
          class="task-card__close"
          type="button"
          @click="closeDialog"
      />

<!--      Блок ввода имени и удаления задачи-->
```

Диалог закрывается по клику на фон или нажатию Esc. Логика обработчика:

```html
<script setup>
import { ref, onMounted } from 'vue'
import { useRouter } from 'vue-router'

const router = useRouter()

const dialog = ref(null)

onMounted(() => {
  // Фокусируем на диалоговом окне, чтобы сработала клавиша Esc без дополнительного клика на окне
  dialog.value.focus()
})

function closeDialog () {
  // Закрытие диалога, всего лишь переход на корневой маршрут
  router.push('/')
}
</script>
```

Атрибуты `tabindex` и `ref` на корневом элементе обеспечивают автоматическую фокусировку на окне при открытии, что позволяет клавише Esc работать без предварительного клика. Попробуйте убрать этот код и проверьте, как изменится поведение Esc.

# Шаг 3. Написание вспомогательных функций

Для дальнейшей работы нам понадобятся две вспомогательные функции: `createUUIDv4` и `createNewDate`.

`UUID` будет служить временным идентификатором для новых задач до их сохранения на сервере. Подробнее о нём можно прочитать в [статье про универсальный уникальный идентификатор](https://ru.wikipedia.org/wiki/UUID).

Добавьте функции в файл `src/common/helpers`:

```javascript
export const createUUIDv4 = () => {
  return 'xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx'.replace(/[xy]/g, c => {
    const r = Math.random() * 16 | 0, v = c === 'x' ? r : (r & 0x3 | 0x8);
    return v.toString(16);
  });
};

export const createNewDate = () => {
  return new Date(new Date().setHours(23,59,59,999));
};
```

# Шаг 7. Компонент выбора пользователя

Выделим логику выбора исполнителя задачи в отдельный компонент. Создайте `TaskCardCreatorUserSelector` в директории `src/modules/tasks/components`.

Предварительно создадим директиву `click-outside` для обработки кликов за пределами компонента. Создайте файл `directives.js` в директории `src/common`:

```javascript
export const clickOutside = {
  mounted(el, binding) {
    el.clickOutsideEvent = function(event) {
      if (!(el === event.target || el.contains(event.target))) {
        binding.value(event, el);
      }
    };
    document.body.addEventListener('click', el.clickOutsideEvent);
  },
  unmounted(el) {
    document.body.removeEventListener('click', el.clickOutsideEvent);
  }
}
```

Зарегистрируйте директиву в `main.js`:

```javascript
import { clickOutside } from './common/directives'

app.directive('click-outside', clickOutside)
```

# Шаг 10. Создание компонента создания и отображения подзадач

Компонент списка подзадач мы уже создали ранее. Теперь можно переиспользовать его:

```html

<!--        Список подзадач-->
        <task-card-view-ticks-list

            @removeTick="removeTick"
```

Обработчики событий:

```javascript
import TaskCardViewTicksList from './TaskCardViewTicksList.vue'
import { createUUIDv4, createNewDate } from '@/common/helpers'
// ...
const createNewTick = () => ({
  // Добавляем временный идентификатор до момента отправки на сервер
  uuid: createUUIDv4(),
  taskId: null,
  text: '',
  done: false
})

//...

function createTick () {
  task.value.ticks.push(createNewTick())
}

// Используем uuid для новых подзадач, id для существующих
function updateTick (tick) {
  const index = task.value.ticks
      .findIndex(({ uuid, id }) => {
        if (uuid) {
          return tick.uuid === uuid
        }
        if (id) {
          return tick.id === id
        }
        return false
      })
  if (~index) {
    task.value.ticks.splice(index, 1, tick)
  }
}

function removeTick ({ uuid, id }) {
  if (uuid) {
    task.value.ticks = task.value.ticks.filter(tick => tick.uuid !== uuid)
  }
  if (id) {
    task.value.ticks = task.value.ticks.filter(tick => tick.id !== id)
  }
}
```

Обработчики:

```html
<script setup>
import { ref, onMounted, watch } from 'vue'
import AppButton from '@/common/components/AppButton.vue'
import { validateFields } from '@/common/validator'
// ...
const isFormValid = ref(true)
// ...

// Отслеживаем изменения в задаче, чтобы сбросить ошибки валидации
watch(task, () => {
  isFormValid.value = true
  validations.value = setEmptyValidations()
}, { deep: true })

function submit () {
  // Валидируем задачу
  if (!validateFields(task.value, validations.value)) {
    isFormValid.value = false
    return
  }
  if (props.taskToEdit) {
    // Редактируемая задача
    emits('editTask', task.value)
  } else {
    // Новая задача
    emits('addTask', task.value)
  }
  // Переход на главную страницу
  router.push('/')
}
</script>
```
