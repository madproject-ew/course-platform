# Файл vite.config.js

`vite.config.js` — файл конфигурации проекта. Разберём назначение его опций и параметров.

```javascript
import { defineConfig } from 'vite'
import vue from '@vitejs/plugin-vue'
import * as path from 'path'

export default defineConfig({
  plugins: [vue()],

  resolve: {
    alias: {
      '@': path.resolve(__dirname, 'src')
    }
  }
})
```

`plugins` — набор плагинов, необходимых для корректной работы сборщика. Поскольку Vite не предназначен исключительно для Vue.js, эта опция необходима. Здесь мы передаём экземпляр официального плагина Vite для работы с Vue.js. При необходимости список плагинов можно расширить в зависимости от потребностей проекта.

`resolve.alias` — определяет, каким путям соответствует алиас `@` в нашем случае. Более подробно мы рассматривали это на этапе настройки проекта.
