# Задание: добавление анимаций в проект OnlinePizza

Анимации делают интерфейс живым и отзывчивым. На этом этапе вы добавите несколько типов анимаций в приложение OnlinePizza.

## Шаги выполнения

### 1. Анимация переходов между маршрутами

Откройте компонент, в котором расположен `<router-view>`. Оберните его в компонент `<Transition>` (или `<RouterView>` с использованием слота) и добавьте CSS-анимацию плавного появления (fade). При переходе между страницами текущий контент должен плавно исчезать, а новый — плавно появляться.

### 2. Анимация элементов корзины

Откройте `src/views/CartView.vue`. Список пицц в корзине оберните в `<TransitionGroup>`. При добавлении новой пиццы она должна плавно появляться со сдвигом (slide), а при удалении — плавно исчезать. Не забудьте добавить атрибут `key` для каждого элемента списка.

### 3. Анимация выбора ингредиентов

В компоненте `IngredientsSelector` (или в том компоненте, который отображает карточки ингредиентов) добавьте анимацию масштабирования (scale). Когда количество ингредиента меняется с 0 на 1 (то есть ингредиент впервые добавляется), карточка ингредиента должна кратковременно увеличиться и вернуться к исходному размеру.

### 4. Анимация состояния авторизации в шапке

Откройте `src/components/AppHeader.vue`. Добавьте плавный переход при смене состояния авторизации: когда ссылка «Войти» заменяется на имя пользователя (и наоборот), переход должен быть анимированным, а не мгновенным. Используйте `<Transition>` с режимом `mode="out-in"`.

### 5. Выделите стили анимаций в отдельный файл

Создайте файл `src/assets/styles/transitions.css`. Перенесите в него все CSS-классы анимаций (`.fade-enter-active`, `.fade-leave-active`, `.slide-enter-from`, `.scale-enter-active` и т.д.). Импортируйте этот файл в `main.js` или в основной файл стилей, чтобы анимации были доступны глобально.

## Проверка

Пройдите по приложению и убедитесь:

- при навигации между страницами виден плавный fade-эффект;
- при добавлении пиццы в корзину новый элемент списка появляется с анимацией slide;
- при выборе ингредиента впервые карточка кратковременно увеличивается (scale);
- при входе и выходе из системы содержимое шапки меняется плавно.
