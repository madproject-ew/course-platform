# Задание: добавление геттеров хранилища в проект OnlinePizza

Геттеры — это вычисляемые свойства хранилища. Они позволяют получать производные данные из состояния без дублирования логики в компонентах. На этом этапе вы добавите геттеры в хранилища конструктора и корзины.

## Шаги выполнения

### 1. Геттеры хранилища конструктора пиццы

Откройте `src/stores/pizza.js` и добавьте следующие вычисляемые свойства (в Composition API стиле это `computed`):

- **`pizzaPrice`** — рассчитывает стоимость текущей пиццы в конструкторе. Формула: цена теста + (цена соуса × множитель размера) + сумма по каждому ингредиенту (цена ингредиента × его количество × множитель размера). Если какой-то из компонентов не выбран, учитывайте это в расчёте.

- **`selectedIngredientsList`** — возвращает массив ингредиентов, количество которых больше нуля. Каждый элемент должен содержать данные ингредиента и его текущее количество.

- **`isPizzaReady`** — возвращает `true`, если выбраны тесто, размер, соус и добавлен хотя бы один ингредиент. Этот геттер пригодится для блокировки кнопки добавления в корзину.

### 2. Геттеры хранилища корзины

Откройте `src/stores/cart.js` и добавьте:

- **`totalPizzasPrice`** — сумма стоимости всех пицц в корзине (цена каждой пиццы × её количество).

- **`totalExtrasPrice`** — сумма стоимости дополнительных товаров (цена каждого товара × его количество).

- **`totalPrice`** — общая стоимость заказа: `totalPizzasPrice` + `totalExtrasPrice`.

- **`cartIsEmpty`** — возвращает `true`, если в корзине нет ни пицц, ни дополнительных товаров.

- **`itemsCount`** — общее количество позиций в корзине (суммарное количество пицц + суммарное количество дополнительных товаров).

## Проверка

Откройте Vue DevTools и проверьте:

- геттеры отображаются в каждом хранилище;
- `pizzaPrice` корректно пересчитывается при изменении выбранного теста, размера, соуса или количества ингредиентов;
- `totalPrice` обновляется при добавлении или удалении товаров из корзины;
- `cartIsEmpty` возвращает `true`, когда корзина пуста, и `false`, когда в ней есть хотя бы один элемент.
