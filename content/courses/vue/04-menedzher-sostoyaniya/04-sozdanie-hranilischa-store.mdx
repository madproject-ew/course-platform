# Создание хранилища Store

Чтобы начать работу с Pinia, необходимо определить хранилище данных через функцию `defineStore`:

```javascript
import { defineStore } from 'pinia'

export const useTodosStore = defineStore('todos', {
  state: () => (
    {
      todos: []
    }
  ),
  getters: {
    getTodos: (state) => state.todos,
  },
  actions: {
    addTodo(todo) {
      this.todos.push(todo)
    },
  },
})
```

Функция `defineStore` принимает два аргумента:

1.  Уникальный идентификатор хранилища.
2.  Объект со свойствами хранилища:

*   state — состояние, то есть данные хранилища;
*   actions — методы для изменения состояния;
*   getters — вычисляемые свойства для извлечения данных.

Существует и альтернативный синтаксис объявления хранилища:

```javascript
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'

export const useTodosStore = defineStore('todos', () => {
  const todos = ref([])
  const getTodos = computed(() => todos.value)
  function addTodo(todo) {
    todos.value.push(todo)
  }

  return { todos, getTodos, addTodo }
})
```

В этом варианте переменные `ref` превращаются в свойства `state`, `computed` — в `getters`, а функции — в `actions`.

## Использование хранилища в компонентах

Объявленное хранилище можно импортировать и использовать в любом компоненте приложения:

```javascript
import { useTodosStore } from '@/stores/todos'

const store = useTodosStore()

const todos = store.getTodos

const newTodo = {
  task: 'Сделать что-то',
  isCompleted: false
}

store.addTodo(newTodo)
```
